Sistema operativo:
Es el software principal que administra los recursos del computador y permite que los programas funcionen correctamente.
Controla procesos, memoria y la comunicación entre el hardware y las aplicaciones.

Node.js:
Es un entorno que permite ejecutar código JavaScript fuera del navegador.
Se utiliza para crear servidores y aplicaciones web que pueden responder solicitudes de usuarios.

Aplicación server.js:
Es el archivo donde se encuentra configurado el servidor de la aplicación.
Define las rutas, las respuestas y las funciones que ejecutará cuando reciba solicitudes.

Puerto:
Es un punto de comunicación que permite que un servidor reciba conexiones.
En esta actividad el servidor utiliza el puerto 3000 para atender las solicitudes.

Logs:
Son registros donde se guarda información sobre la actividad del servidor.
Permiten identificar errores, revisar eventos y monitorear el funcionamiento del sistema.
